# Development and Maintenance Guide 

Цей файл містить інструкції для розробників та підтримки проекту "Study Less Pocket".

## 1. Оновлення контенту зі слайдів
Коли з'являються нові презентації, використовуйте промпт із файлу:
`prompts/Prompt_parsing_slides.txt`

Він допоможе перетворити слайди на структуровані Markdown-файли, які потрібно зберігати в папці `documents/`.

## 2. Оновлення навчальних промптів (Base Knowledge)
Після додавання нових уроків у папку `documents/`, необхідно оновити файли `Prompt_Part1.txt` та `Prompt_Part2.txt` в папці `prompts/`.

**Інструкція для автоматичного оновлення:**
 `prompts/Prompt_Generator_Tool.md`

**Команда для Gemini/AI:**
"Виконай оновлення промптів згідно з інструкцією в prompts/Prompt_Generator_Tool.md, забезпечивши 100% покриття словника та вправ."

## 3. Генерація Linguist-DSL для застосунку
Цей інструмент призначений для підготовки контенту для мобільного/веб застосунку у форматі DSL.

**Інструкція для використання:**
 `prompts/Linguist_DSL_Generator.md`

**Команда для Gemini/AI:**
"Ознайомся з інструкцією в prompts/Linguist_DSL_Generator.md і запитай мене, які файли потрібно обробити для додавання в dsl.txt."

**Важливо:** 
- Результат завжди додається (append) до файлу `dsl.txt`.
- Кожен урок відокремлюється символом `---`.
- Блок `[map:]` генерується лише для розділу Vocabulary.

## 4. Додавання нової інформації (Vocabulary Hooks, Puns, Rules)
Цей алгоритм використовується для точкового додавання нових прикладів, каламбурів або правил до існуючих файлів у папці `documents/`.

**Інструкція для використання:**
 `prompts/Content_Addition_Workflow.md`

**Команда для Gemini/AI:**
"Використовуй алгоритм додавання інформації з prompts/Content_Addition_Workflow.md для оновлення файлів у documents/."
