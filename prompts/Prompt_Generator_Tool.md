# ІНСТРУКЦІЯ ДЛЯ ОНОВЛЕННЯ НАВЧАЛЬНИХ ПРОМПТІВ (Meta-Prompt)

Цей файл призначений для Gemini CLI або іншої LLM з доступом до файлів проекту.

## ЗАВДАННЯ:
Просканувати папку `documents/`, витягнути ПОВНУ базу знань та оновити два текстові файли-промпти.

## АЛГОРИТМ ЗБОРУ (Критично для глибини тестування):
Для кожного `.md` файлу в `documents/` витягуй:
1. **Vocab (100% покриття):** Виписуй КОЖНУ пару "слово — переклад" з таблиць словника. Нічого не скорочуй.
2. **Grammar Essentials:** Всі формули та всі приклади речень, що пояснюють правило.
3. **Practice & Translation (Пріоритет):** Виписуй УСІ готові речення з вправ (Written Practice) та блоків перекладу (Translation Blocks). Це готові завдання для тестування.

## СТРУКТУРА ФАЙЛІВ-ПРОМПТІВ:
Кожен файл (`Prompt_Part1.txt`, `Prompt_Part2.txt`) повинен починатися з інструкції для вчителя:
- "Твоя мета — переконатися, що учень засвоїв КОЖНЕ слово та КОЖНУ фразу з обраної теми."
- "Проводь тестування, поки не пройдешся по всьому списку словника та прикладів."
- "Постійно вимагай використання Polite English softeners."

## ЯК ЗАПУСТИТИ:
Напиши мені: "Онови промпти згідно з інструкцією в Prompt_Generator_Tool.md, забезпечивши 100% покриття словника".
